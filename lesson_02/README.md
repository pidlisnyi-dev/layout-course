# Lesson 02 — Project File Structure, Introduction to HTML

## What I learned
- HTML syntax: tags with closing elements and self-closing tags
- HTML attributes and their basic structure
- Structural tags: header, main, footer, div
- Content tags: h1–h6, p, ul/ol, li, a, img, span, mark, br, abbr
- Element types: block, inline, block-inline
- Basic rules of HTML nesting
- Basic attributes: class, id, title, target, src, alt, href

## Practice
- A formatted article about Subnautica 2 with a complete HTML structure
- Navigation implemented via a list with anchor links (`ul > li > a`)
- Table of contents added using `ol` with quick navigation between sections
- `table` used for structured data (biomes, transportation)
- Nested lists implemented in the section on psychological triggers
- `abbr` used for the terms NPC and environmental storytelling
- Header with logo (`a > img`) and footer with copyright created

## Result
A multi-section HTML page with structure, navigation, and various content types has been created.
